## Описание проекта

«Умная таблица» — это интерактивная веб‑таблица для отображения и фильтрации данных. Проект реализует базовые функции работы с данными:
- пагинация;
- сортировка по столбцам;
- фильтрация по полям (включая выпадающий список продавцов);
- поиск;
- сброс фильтров.

## Функциональные возможности

1. **Пагинация**  
   - Переключение между страницами.
   - Выбор количества строк на странице (10, 25, 50, 100).

2. **Сортировка**  
   - По дате (возрастание/убывание).
   - По сумме (возрастание/убывание).

3. **Фильтрация**  
   - По клиенту (текстовый ввод).
   - По продавцу (выпадающий список).
   - По сумме (диапазон: от и до).

4. **Поиск**  
   - Глобальный поиск по всем полям таблицы.

5. **Сброс фильтров**  
   - Кнопка «Reset all filters» для очистки всех полей.
